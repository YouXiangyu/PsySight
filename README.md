# PsySight (心理委员俺不得劲)

**创建时间**: 2026-01-14 | **更新时间**: 2026-03-06  
**版本**: v5.0 | **状态**: 全量计划功能已实现（待集成验收）

## 1. 项目概述

**PsySight** 是面向大学生的智能心理健康助理系统。系统提供倾诉对话、专业量表、情绪辅助分析、心理报告与危机守护能力，帮助用户在低门槛、低压力的环境中完成心理自助筛查与支持。

> **⚠️ 免责声明**  
> 本系统仅用于心理健康筛查、情绪支持与自助建议，不构成医疗诊断或治疗方案。若存在自伤/自杀风险，请立即联系专业机构与紧急热线。

## 2. 当前实现能力

### 2.1 智能倾诉空间
- 共情对话（DeepSeek R1）+ 规则兜底推荐量表。
- 登录用户支持历史会话列表与上下文恢复。
- 每条 AI 回复支持 👍/👎 反馈记录。
- 危机关键词触发强提醒求助卡片（会话结束前持续显示）。

### 2.2 身份识别与匿名体验
- 邮箱+密码注册/登录/登出（密码哈希存储）。
- 未登录可匿名聊天。
- 匿名测评提交前提示登录以保存报告与历史。

### 2.3 安全守护
- 全局固定“紧急求助”入口。
- 内置热线列表（至少 2 条）。
- PHQ-9/GAD-7 严重分值时，报告顶部优先展示专业求助建议。

### 2.4 专业评测与多模态情绪
- 量表库分类展示，含题目数与预计用时。
- “AI 帮你选”推荐入口。
- 作答过程支持进度、预计剩余时间、暂停与继续。
- 人脸情绪采集为**可选能力**：默认关闭，用户同意后开启，仅上传聚合情绪特征。

### 2.5 心理报告
- 报告包含总分、严重程度、分值解释、个性化建议。
- 展示“答题期间情绪分布”可视化条形图（开启情绪采集时）。
- 报告页支持浏览器打印导出 PDF。

### 2.6 创意表达与投射分析
- 画板支持画笔、橡皮、颜色、清空、撤销。
- 支持房树人引导勾选与自述补充。
- 后端基于绘画元数据 + 文本描述生成温暖解读。

### 2.7 群体共鸣与研究导出
- 首页/报告页展示匿名群体统计（标注基于 N 位用户）。
- 提供管理员令牌导出接口（CSV/JSON）与导出审计日志。

## 3. 技术栈

- 前端：Next.js 14 (App Router), TypeScript, Tailwind CSS, face-api.js
- 后端：Flask, SQLAlchemy, SQLite, DeepSeek API
- 架构：前后端分离（Next 重写代理 `/api` -> Flask）

## 4. 本地开发

### 4.1 后端
```bash
cd backend
pip install -r requirements.txt
copy .env.example .env
python seed.py
python app.py   # 默认端口 8004
```

### 4.2 前端
```bash
cd frontend
npm install
copy .env.example .env.local
npm run dev     # 默认端口 8003
```

## 5. 关键环境变量

### 后端
- `DEEPSEEK_API_KEY`
- `DEEPSEEK_MODEL`（默认 `deepseek-reasoner`）
- `ADMIN_EXPORT_TOKEN`
- `DATABASE_URL`
- `SECRET_KEY`
- `FRONTEND_ORIGIN`

### 前端
- `NEXT_PUBLIC_API_BASE_URL`（默认 `/api`）
- `BACKEND_URL`（Next 重写代理目标）
