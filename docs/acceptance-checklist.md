# PsySight 全量验收清单

**创建时间**: 2026-03-05 | **更新时间**: 2026-03-05

## 1. 环境与启动
- [ ] 后端 `pip install -r requirements.txt` 成功，无依赖缺失。
- [ ] 前端 `npm install` 成功，无关键依赖冲突。
- [ ] 前端 `npm run build` 成功通过。
- [ ] 后端可正常启动在 `8004`，前端可正常启动在 `8003`。
- [ ] 配置 `.env` 后可成功调用 DeepSeek。

## 2. 智能倾诉空间（史诗1）
- [ ] 首页可在 2 秒内输入消息。
- [ ] AI 能输出共情回复并引用用户上下文要点。
- [ ] 对话可推荐量表卡片并可跳转测评页。
- [ ] 登录后显示历史会话列表，点击可恢复上下文。
- [ ] 每条 AI 消息可进行 👍/👎 反馈并成功写入。

## 3. 身份识别（史诗2）
- [ ] 注册流程不超过 3 步（邮箱/密码/可选昵称）。
- [ ] 密码以哈希方式存储（非明文）。
- [ ] 匿名状态可直接聊天。
- [ ] 匿名提交测评前弹出“登录保存报告”提示。

## 4. 安全守护（史诗3）
- [ ] 输入危机关键词后，对话出现紧急求助卡片。
- [ ] 求助卡片显示至少 2 条可拨打热线。
- [ ] 所有页面可见全局“紧急求助”按钮。
- [ ] PHQ-9>=20 或 GAD-7>=15 时，报告顶部出现专业求助建议。

## 5. 专业评测 + 多模态情绪（史诗4/5）
- [ ] `/scales` 页面按分类展示量表，显示题目数与预计时长。
- [ ] “AI 帮你选”能根据描述给出推荐量表。
- [ ] 作答页面显示进度、剩余时间、暂停/继续。
- [ ] 摄像头默认关闭，开启前弹出隐私同意说明。
- [ ] 关闭情绪采集后前端内存情绪数据被清空。
- [ ] 报告页可展示答题期间情绪分布图。

## 6. 创意表达与投射分析（史诗6）
- [ ] 画板支持画笔、橡皮、颜色、清空、撤销。
- [ ] 房树人引导模式可勾选完成元素。
- [ ] 提交后返回温暖解读（元素观察+心理映射+建议+鼓励语）。

## 7. 心理报告（史诗7）
- [ ] 报告显示总分、严重程度、分值解释。
- [ ] 报告包含 2-3 条可执行个性化建议。
- [ ] 报告可通过浏览器打印导出 PDF。
- [ ] 底部包含非医疗免责声明。

## 8. 治愈感设计 + 群体共鸣（史诗8/9）
- [ ] 量表提交后出现 2-3 秒鼓励过渡页。
- [ ] 首页/报告展示“基于 N 位用户”的匿名统计。
- [ ] 管理员令牌导出接口支持 JSON/CSV。
- [ ] 导出数据中不包含用户 ID 和 IP。

## 9. 稳定性与可用性（史诗10）
- [ ] 关键页面在 375px-1440px 均可用。
- [ ] API 错误会显示友好提示而非白屏。
- [ ] 聊天输入断网后会本地暂存，恢复后可继续发送。
- [ ] `/api/health` 与 `/api/metrics/availability` 可返回可观测性数据。
